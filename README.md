##  RentaMovil — App Móvil
> *Alquila fácil, conduce libre*

Aplicación móvil del sistema de alquiler de vehículos **RentaMovil**, desarrollada con **React Native + Expo** para Android 13 y 14.

---

##  Tecnologías

- React Native 0.81.5
- Expo ~54
- Expo Router — Navegación por archivos
- React Navigation — Bottom tabs
- TypeScript
- Expo Vector Icons

---

##  Instalación

```bash
git clone https://github.com/tu-org/rentamovil-frontend.git
cd appmovilrenta
npm install
npm run android
```

---

##  Scripts

```bash
npm run start       # Iniciar servidor Expo
npm run android     # Ejecutar en Android
npm run web         # Ejecutar en navegador
npm run lint        # Revisión de código
npm run reset-project  # Limpiar y resetear proyecto
```

---

##  Estructura
appmovilrenta/

├── components/       # Componentes reutilizables
├── constants/        # Tema, colores
